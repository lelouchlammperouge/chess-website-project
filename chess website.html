Chess is a very fascinating game which requires a lot of visual representation.
Using visual representation you are able to plan ahead multiple moves in a game of chess.
Chess also requires a huge amount of calculations each move, this is becasue there are
practically quadrillions of different combinations in just a few moves of chess.
These combination makes many high level chess games very unique with a battle of the
minds.
